# 📖 About

**GradeVision** is a modern academic companion designed specifically for university students.

Unlike traditional GPA calculators, GradeVision provides **intelligent academic insights**, **performance analytics**, and **grade prediction tools** to help students understand their progress and improve their academic performance before it's too late.

Whether you're monitoring attendance, analyzing class test performance, simulating final exam scores, or organizing your academic routine, GradeVision keeps everything in one beautiful and intuitive application.

## 🧠 Smart Prediction Engine

GradeVision continuously analyzes your academic data and warns you before problems become serious.

### Attendance Alerts
- Detects attendance below university requirements.

### Weak Class Test Alerts
- Identifies poor CT performance.

### Weak Lab Performance
- Detects low sessional/lab assessment scores.

### Section Performance Analysis
- Finds significant differences between Section A and Section B.

### Downward Trend Detection
- Detects declining performance over time.

### Final Exam Safety Buffer
- Warns when passing requires extremely high final exam marks.
